                            PloneShellConsole
================================================================================

PloneShellConsole is a Plone product, which intends to provide an web interface
to manage, execute, and track shell scripts in any remote machine.  So you can
write shell scripts for a remote machine on your Plone site, and then execute
those scripts, and review the logs.

It meant to provide a secure console to manage remote machine.

A good example will be execute mvn, collect mvn output, anylyze it.
